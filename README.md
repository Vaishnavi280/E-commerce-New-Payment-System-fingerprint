# Django-E-commerce-Website

## Create Python Virtual Environment

### On mac-os and Linix:

`python3 -m pip install --user virtualenv`

### On Windows:

`py -m pip install --user virtualenv`

### On mac-os and Linix:

`python3 -m venv env`

### On Windows:

`py -m venv env`

## Activate Environment

### On mac-os and Linix:

`source env/bin/activate`

### On Windows:

`cd env`  
`Scripts\activate`
